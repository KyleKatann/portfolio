PROGRAMS

python用公開モジュール「oneDpack」

実装言語：python・高速数理最適化線形／整数計画ソルバーであるgurobiかmypulp
pip install oneDpackでインストール、一次元binpacking問題を解くことが可能。
packing関数の第一引数にbinの大きさを、第二引数にitemのiterbleを入れる。
oneDpack.packingは問題を最短路問題に帰結し、binの大きさが小さい時に高速に解ける。
oneDpack.packingCGは列生成法を使用し、itemの数が少ない時に高速に解ける。
github:https://github.com/KyleKatann/oneDpack

PROGRAMS

倉庫内ピッキングTSP問題用ソフトウェア「imagine_warehouse」

実装言語：python・高速数理最適化線形／整数計画ソルバーであるgurobi
35点のnodeで表現された倉庫内のピッキングTSP問題が解ける。
具体的に、15番から出発し、ABCD...と名のついた商品を最短路でピッキングして、15番に帰ってくるルートを表示する。
倉庫内の表現はwarehouse_node.xlsxで規定、ABCD...がどこにあるかはwhere_is_inventry.xlsxで規定する。中間計算結果としてwarehouse_root.xlsx、queryとしてquery.xlsxを自動生成する。
warehouse_node.xlsxさえ改変すれば現実の倉庫にも対応できる。
github:https://github.com/KyleKatann/imagine_warehouse heroku:https://guarded-brook-03527.herokuapp.com
